# Workflow

This helper supports the handoff phase when a LaTeX manuscript must be prepared for a Word-based journal process.

Recommended workflow:

1. Scan the LaTeX project.
2. Read `conversion_plan.md`.
3. Fix unresolved references and missing figures.
4. Convert body text through your preferred route.
5. Rebuild equations, captions, cross-references, and bibliography in Word.
6. Use `journal_handoff.docx` as a coauthor/editor checklist.

The generated `.docx` is a checklist, not a full converted manuscript.
